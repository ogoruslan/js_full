# Слайдер з котиками — HW 11

Домашнє завдання містить **дві версії** слайдера, кожна у своїй гілці:

| Гілка | Підхід | Файл |
|---|---|---|
| `prototype-version` | Конструктор + `prototype`, наслідування через прототипний ланцюжок | `slider.js` |
| `class-version` | ES6 `class`, наслідування через `extends`, + автопауза на hover | `slider.js` |

`index.html` та `styles.css` однакові в обох гілках — відрізняється лише
реалізація `slider.js`.

## Як подивитись кожну версію

```bash
git checkout prototype-version   # варіант на прототипах
git checkout class-version       # варіант на класах
```

Потім просто відкрийте `index.html` у браузері.
