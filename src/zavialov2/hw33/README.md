# Movie Searcher

A simple web application for searching movies using the OMDb API.

## Features

- Search movies by title
- View detailed movie information
- SCSS compilation with Gulp
- BrowserSync live reload
- Autoprefixer
- CSS minification

## Technologies

- HTML
- SCSS
- JavaScript
- Gulp
- OMDb API

## Installation

```bash
npm install
```

## Run

```bash
npm start
```

## Build

```bash
npm run build
```
