# Домашня робота №6

## Опис проекту
У цій роботі було відтворено відому абстрактну композицію **Піта Мондріана** засобами HTML та CSS.

## Технології та методи
Для досягнення максимальної схожості з оригіналом та точного позиціонування елементів було використано:
*   **Складна система вкладених таблиць** (`<table>`, `<tr>`, `<td>`) — для побудови сітки картини.
*   **Вбудований CSS** (тег `<style>` у секції `<head>`) — для визначення основних параметрів таблиць та комірок.
*   **Inline-стилі** (атрибут `style=""`) — для точкового керування шириною та висотою окремих блоків.
